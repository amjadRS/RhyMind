# RhyMind

RhyMind is a Kotlin-based Android application designed to help users manage reminders and coordinate meetings more efficiently using location-based travel time estimation.

The app calculates how long it takes to reach a destination and suggests the best time to leave, helping users avoid being late. It also supports group coordination where each participant receives a personalized leave time based on their location.

---

## Features

- Smart reminder management
- Location-based travel time calculation
- Leave-time suggestions based on real travel time
- Push notifications
- Group meeting coordination
- Friend system for shared meeting planning

---

## Technologies Used

- Kotlin
- Android SDK
- Firebase
- Firebase Cloud Messaging (Push Notifications)
- Google Maps API
- Google Places API
- Retrofit
- MVVM Architecture

---

## How to Run the Project

1. Clone the repository

2. Open the project in **Android Studio**

3. Add your API keys to `local.properties`

4. Sync Gradle

5. Run the application on an **emulator or physical device**

---

## Project Idea

RhyMind helps users avoid being late by calculating travel time to a destination and suggesting the optimal time to leave.

The application also supports **group meeting coordination**, where each participant receives a customized leave time based on their own location and the shared meeting point.
