# RhyMind

RhyMind is an Android application built with Kotlin that helps users manage reminders and coordinate meetings more effectively using location-based travel time estimation.

## Features
- Smart reminder management
- Location-based travel time calculation
- Push notifications
- Meeting coordination for individuals and groups
- Leave-time suggestions based on user location and destination
- Friend/group functionality for shared meeting planning

## Technologies Used
- Kotlin
- Android SDK
- Firebase
- Google Maps API
- Google Places API
- Push Notifications
- Retrofit
- MVVM Architecture

## Idea
The app helps users avoid being late by calculating how long it takes to reach a destination and suggesting the right time to leave. It also supports group coordination, where each participant can get a personalized leave time based on their own location.

## Setup
1. Clone the repository
2. Add your API keys to `local.properties`
3. Sync Gradle
4. Run the app
